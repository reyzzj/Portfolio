Bill Please mobile application

A simple app to help calculate and split cost of food with friends with different variables that affect the total food cost inputs of the amount of money, amount of people, GST, service charge(SVS), discounts, payment method. The user then can just press split to split the bill evenly amonst the amount of people.

Thought Process

Since it was very simple app to make, some things that made up this such as

Radio Group -

Radio Group is needed to only allow the payment method select 1(either Cash,Paynow,PayLah)

Radio Button -

Allows the user to select their payment method

Toggle Button

Allows the user to toggle GST and SVS to see if the eatery require these additional charges

Text View

Basic view for all the text

Edit Text

Allows the user to input text for the output calculation

Toast

Will appear to warn the user if a required field is empty(e.g. amount/pax)
